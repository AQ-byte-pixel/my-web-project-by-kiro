# 简单网页系统

现代化的静态网页，采用HTML5、CSS3和响应式设计。

## 项目结构

```
simple-webpage/
├── index.html          # 主HTML文件
├── styles/
│   └── main.css        # 主样式表
├── scripts/
│   └── main.js         # JavaScript文件
├── assets/
│   └── images/         # 图片资源
├── tests/              # 测试文件
└── package.json        # 项目配置
```

## 开发环境设置

1. 安装依赖：

```bash
npm install
```

2. 运行测试：

```bash
npm test
```

3. 启动本地服务器：

```bash
npm run serve
```

4. 验证HTML：

```bash
npm run validate-html
```

## 技术栈

- HTML5 - 语义化标记
- CSS3 - 现代样式和布局
- JavaScript - 可选的交互增强
- Jest - 测试框架
- fast-check - 基于属性的测试

## 特性

- 响应式设计
- 语义化HTML结构
- 现代CSS布局
- 可访问性优化
- 跨浏览器兼容性
